# Pet Tinder

### Features
- Register and sign in to account
- Verify profile with e-mail verification
- Reset password and delete account
- Browse pets from API
- Like, reject pets and store this data in database
- Filter pets

### Tech stack
- HTML 5
- CSS 3
- SCSS
- Javascript ES6
- Typescript
- React
- Redux
- Firebase
